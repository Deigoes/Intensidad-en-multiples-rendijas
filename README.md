# Intensidad en interferencia de varias fuentes síncronas
Este repositorio contiene una simulación numérica del patrón de interferencia y difracción producido por un sistema de N rendijas. El proyecto está diseñado de forma modular, separando la lógica física de la visualización y la configuración. El objetivo es visualizar la evolución de la distribución de intensidad I/I_0 conforme aumenta el número de rendijas (N).
